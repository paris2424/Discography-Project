Discography Website - Revamp: Style Guide

A Guide for the proper styling of web design elements on the Discography Website Revamp by David Johnson.


Background:

All pages use the image texture of dark wood with a lighter radial gradient in the center.



Colors:

All pages use the following color palette: (Colors developed with Adobe Kuler)

Blue: Hex#6699FF
Orange: Hex#E87B0C
Black: Hex#000
White: Hex#FFF



Typography:


Font

All pages use Andalé Mono, a monospaced sans-serif typeface designed by Steve Matteson for terminal emulation and software development environments, reversed-out in white. (See Wikipedia)


Header (h1): color: #fff, font-size: 1.75em


Header (h2): color: #fff, font-size: 1.50em

Paragraph (p) and Lists (li): margin: 2%, color: #fff, font-size: 100%, line-height: 1.5em



Buttons:

Form uses the following button shape and color style:SEND(background-color: #6699FF, border-radius: 6px, padding: 6px, margin: 6px, color: #fff, font-family: 'Andale Mono')



Citations and Hot Links:
